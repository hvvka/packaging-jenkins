=======
Strings
=======

A place for implementation of string algorithms

Features
--------

* To see all the available functions in a module, you can just type ``help()`` with the module name as argument. For example,

.. code-block:: python

    >>> from lab10 import strings
    >>> help(strings)
        Help on package lab10.strings in lab10:

        NAME
            lab10.strings - Collection of string methods and functions

        PACKAGE CONTENTS
            anagram
            palindrome
